# Changelog

## 0.4.1-free.0 - 2025-10-08
Initial release for the free edition of [KRaster Importer] at v0.4.1.

[KRaster Importer]: https://assetstore.unity.com/packages/slug/227895

### Added
- Thumbnails: See Source Image assets in the editor via internal merged image.
- Source Image Preview: See the internal merged image in the inspector.
- Manual Image Converter: Quick conversions to PNG on the same folder when right-clicking an image asset.
- Preferences: Options to configure what metadata is shown in the preview.
- Importer, Preferences: Disabled UI and error boxes and dialogs for the unavailable features which would be in the full version, for demo purposes, with a [link to the asset store to upgrade][KRaster Importer] and unlock them.
